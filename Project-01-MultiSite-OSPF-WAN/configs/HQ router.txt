interface GigabitEthernet0/0.10

 ip helper-address 10.0.10.1
 no shutdown

interface GigabitEthernet0/0.20

WAN Interfaces

 ip address 172.16.1.1 255.255.255.252

 no shutdown

interface GigabitEthernet0/2
 ip address 172.16.2.1 255.255.255.252

 no shutdown
